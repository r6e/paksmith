# Paksmith

A cross-platform tool for exploring and extracting Unreal Engine game assets. Written in Rust.

## Status

Early development. Project scaffolding in place; implementation beginning with .pak container reading.

## Building

```sh
cargo build --workspace
```

## Running

```sh
cargo run -p paksmith-cli -- <command>
```

## Testing

```sh
cargo test --workspace
```
